# MikroTik_
 Repozitorijs tika izveidots ar VisualStudio palîdzîbu un tur nepievienojâs Debug mapîte
 Tâdçï izveidoju "MikroTik.rar" failu kurâ atrodas Atmel studio gan proteus projekts
 
 
 Programa tika rakstîta izmantojot Microchipstudio, bet programmas un shçmas pârbaude tika
 pârbaudîta Proteus.
 
 Shçma sastâv no Atmega328P un 10 led gaismas diodes kuras pievienotas pie PORTD 0...7 un PORTB 0,1 izvadiem.
 Shçmâ arî ir tris pogas
 1. poga pievienota pie PORTC 0 izejas. Kas atbild par skrejoðo gaismiòu virziena maiòu.
 2. poga pievienota pie PORTC 1 izejas. Kura palçnina spîdçðanas laiku vienreiz uzspieþot laiks tiek palielinâts par 50ms.
 3. poga pievienota pie PORTC 2 izejas. Kura samazina spîdçðanas laiku vienreiz uzspieþot laiks tiek samazinâts par 50ms.
